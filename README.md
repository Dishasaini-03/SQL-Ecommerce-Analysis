# SQL-Ecommerce-Analysis
## Project Overview
This project contains SQL business queries on an ecommerce database built using PostgresSQL.

 ## Tools Used
- PostgresSQL
- DBeaver

## Database Tables
-customers
-orders
-order_items
-products

## Business Queries
1. Order by Status
2. Total Revenue
3. Top 5 Customers by Spending
4. Best Selling Products
5. Monthly Sales Trend
6. Average Order Value
7. Revenue by Category
8. Pending & Cancelled Orders
9. Customer with Most Orders
